# 许可证

本项目主代码已调整为 **PolyForm Noncommercial 1.0.0**。

这意味着：

- 允许个人学习、研究、评估和内部非商业使用
- 允许在非商业前提下复制、修改和分发
- **不允许将本项目用于商业目的**
- 如需商业使用，需要获得版权所有者的单独书面授权

## 当前授权说明

项目根目录及以下位置的许可证文件已统一：

- [`/LICENSE`](../LICENSE)
- [`/FlyTest_Django/LICENSE`](../FlyTest_Django/LICENSE)
- [`/FlyTest_Vue/LICENSE`](../FlyTest_Vue/LICENSE)
- [`/docs/LICENSE`](./LICENSE)

## 重要说明

PolyForm Noncommercial 不是 OSI 定义的开源许可证，因此如果你希望项目保持“开源且禁止商用”，需要注意：

- “禁止商用”与传统开源定义通常不兼容
- 对外分发前，建议明确商业使用边界和授权方式
- 若后续需要商业授权，建议单独维护商业授权条款

## 第三方依赖许可证

本项目依赖的第三方组件仍然分别遵循它们自己的许可证，常见包括：

- Django / Django REST Framework：BSD 类许可证
- Vue / Vite：MIT 类许可证
- LangChain / OpenAI SDK：各自原始许可证
- Playwright、Qdrant、FastAPI 等：各自原始许可证

这些第三方依赖的许可证并不会因为本项目主代码改为非商用协议而自动改变。  
分发时仍应遵守对应依赖的原始许可证要求。
